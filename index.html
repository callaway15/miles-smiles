Fun facts

    Flight deals via travel-dealz.de

    Open travel-dealz.de ↗
    Live deals are pulled from travel-dealz.de through a public proxy, so this page needs internet. The last results are cached so you still see something offline. All prices & links belong to travel-dealz.de.

    Earning tables (editable)

    ⚠️ Seeded percentages are starting values. Turkish publishes the exact %% per booking class for each airline — confirm/edit a row the first time you use it. Official page: Earn miles as you fly →

    Backup & settings

    All data is stored only in this browser (localStorage). Export regularly so you never lose your history.

    Goal thresholds

    Residence is set per traveller (affects renewal info). Defaults reflect members living outside Türkiye: renew Elite with 25,000 status miles in year 1 (37,500 over 2 years).

    Built as an offline planning aid. Status miles ≈ flight distance × booking-class %. Great-circle distance is a close approximation of Turkish's published flight distance.
